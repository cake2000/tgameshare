Lesson A-2: Bringing Coco Home

[Computer, Core]


Overview

In this lesson, students learn the basics of programming with blocks using Scratch. We continue the theme of brining Coco home, and transfer the knowledge of arrow command programming from lesson 1 to the computer world.

Students will first learn to place arrow blocks through drag-and-drop to send Coco home, and then learn to debug programs with incorrect blocks.

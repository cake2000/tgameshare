Introduction

After a successful breakshot, the 2 players start to make Call Shots, where one target ball and one target pocket are required. For a simple starting point, in this tutorial, we teach your robot to hit the black ball into a pocket. Shall we start?

First let's add a new function this.getCallShot, which specifies the aiming point's coordinate and how much strength to use in a call shot. It should be very similar to your this.getBreakShot function, returning a shot command object with properties aimx/aimy/strength.

When you are done, hit the TEST button to have your robot take its first call shot.

this.getCallShot = function() { return { aimx: 0, aimy: 0, strength: 500, }; };

Now to make it interesting, we are placing the black ball on a random point between the table center and the bottom center pocket. In numbers, the x coordinate of the black ball is always 0, but the y coordinate may be any number between 0 and 400. But don't worry, your robot can retrieve the position of the black ball as follows:

var blackBall = world.Balls[1];

Challenge: please add the line above to the beginning of the this.getCallShot function, then modify the value of aimx/aimy to use blackBall.x and blackBall.y so that your robot can always hit the black ball.

this.getCallShot = function() { var blackBall = world.Balls[1]; return { aimx: blackBall.x, aimy: blackBall.y, strength: 800, }; };
Pocketing the black ball

Now let's try to hit the black ball into the bottom center pocket. To kick the black ball vertically down, the cue ball needs to be on top of the black ball when they touch, as shown below.

When the cue ball is touching the black ball, the distance between their centers is exactly 2 times the ball radius, i.e. the ball diameter, as represented by the variable world.BallDiameter.

Challenge: change aimy to use both blackBall.y and world.BallDiameter so that the cue ball hits the black ball into the bottom center pocket.

this.getCallShot = function() { var blackBall = world.Balls[1]; return { aimx: blackBall.x, aimy: blackBall.y - world.BallDiameter, strength: 800, }; };
Tutorial Completed!

Congratulations on completing your second tutorial! Try to test run it a few more times, and note that your robot can hit the black ball no matter where it appears.

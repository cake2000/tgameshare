Introduction

Advanced and Professional levels have different features and limits, and your robot can decide to behave differently based on the game difficulty level.
Difficulty Levels

The world object contains a field level that tells what level of game you are playing. As shown in the table below, different levels have different limits on strength, and the game engine will cap your strength at these limits.

world.level game level strengh limit
0 Beginner 1000
1 Advanced 1100
2 Professional 1200

Challenge: Please add a switch clause in your function this.getBreakShot to specify different strength depending on the game level.

this.getBreakShot = function() { // use switch clause based on world.level switch (world.level) { case 0: return { aimx: 0, aimy: 0, strength: 1000 }; case 1: return { aimx: 0, aimy: 0, strength: 1200 }; case 2: return { aimx: 0, aimy: 0, strength: 1700 }; } };
Tutorial Completed!

The same idea can be applied to call shots as well, so that your robot can adapt its behavior by checking world.level.

The Break Shot

In this tutorial, we teach your robot how to take a break shot, which breaks the ball pack to start a new game.

Let's start by adding a new function this.getBreakShot to your robot program. The game engine calls this function to get your robot's break shot command.

Please copy or type the code into the robot code editor on the right. When you are done, you can chat me "next" or "ok" to continue.

this.getBreakShot = function() { };

Now let's enhance your break shot function to return an object with 2 properties aimx and aimy, both with a initial value of 0. This will tell the game engine to aim at the center of the table for the shot.

Hint: ask about "table coordinate system" for more information.

When you are done, click the TEST button on the top right to watch your robot's break shot in action!

Note that after the test, you can click the "Exit" button or press the ESC key to dismiss the game window.

this.getBreakShot = function() { return { aimx: 0, aimy: 0 }; };
Hitting yellow ball first

Now I have a new challenge for you. Can you change the value of aimx and aimy so that the cue ball will hit a yellow ball first?

You can try different values of aimx and aimy, and click the "TEST" button to check the result.

this.getBreakShot = function() { return { aimy: 41, aimx: 0, }; };
Taking valid break shots

At this time you probably have found out an issue with your break shot: it is not strong enough. Remember your break shot needs to pocket a colored ball or drive at least 4 balls to the rails.

So far your shot has been using a default strength of 500. You can set the strength by adding a new field strength in the object returned.

Challenge: please specify the shot strength to get a valid break shot.

// new function to specify break shot command this.getBreakShot = function() { return { strength: 1000, aimx: 0, aimy: 0, }; };

Congratulations on completing your first tutorial!

For more exercise, please try to improve your robot's break shot by changing strength/aimx/aimy until you are satisfied. Ideally you want to pocket one or more colored balls (but not the cue ball or the black ball) in the break shot.

Whenever we need randomization, we "call" the function, Math.random, like this: "number = Math.random()". To "call" a function is to execute the code contained in the function. The call to the function includes a list of parameters wrapped by a pair of "()" after the function name. If the parameter list is empty, as in the case for the function Math.random, you need only append "()".

The number returned by Math.random can be any number between 0 and 1, such as 0.251. Each time you call Math.random, it will return a new random number. This is like a magician pulling numbers out of a magic bag. No one knows exactly which number he'll pull out next, except that it won't be less than 0 or more than 1.

Random numbers are very interesting. For example, when we execute 'Math.random()', we get a random number between 0 and 1; when we execute '2 * Math.random()', we get a random number between 0 and 2, essentially doubling its range of possible values; and if we execute '2 * Math.random() + 3', we shift the range of values (in the positive direction along the number axis) by 3, generating a random number between 3 and 5.

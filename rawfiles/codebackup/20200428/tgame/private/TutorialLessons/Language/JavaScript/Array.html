In JavaScript, an array is a collection of similar objects bound together. An array can hold a list of numbers, strings, or other types of objects. For example, like the toy cabinet below, the bears array holds four bear objects. Each bear object can be accessed by an integer (element index) specified in a square bracket.

JavaScript arrays are zero-indexed, which means that the index of the first element of an array is 0. Therefore, bears[0] is the first bear, and bears[3] is the 4th bear.

A JavaScript array is equivalent to in Scratch, and when you use bears[0], it is like reading an item from a list using in Scratch.

The easiest way to create an array is to use an array literal like this, with a pair of square brackets wrapping up a list of items separated by commas: 

bears = ["bear0", "bear1", "bear2", "bear3"];

You can access or modify any element using its index number. 

// get the first element
firstBear = bears[0];
// modify the last element
bears[3] = "Teddy";

You can find out how many elements are contained in an array using its length property like this: 

// four bears in total
numberOfBears = bears.length;

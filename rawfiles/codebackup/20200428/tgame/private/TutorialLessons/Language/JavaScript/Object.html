Before continuing with the tutorial, let's have a review on the concept of object in JavaScript.

An object is used to represent a real-world object, such as a car or a movie, in the computer world.

Just as we describe a real-world object using specific attributes (properties) such as color and shape, a JavaScript object also has its own properties. Each property is associated with a corresponding value, and a ":" is expected between the property name and value. For example, when we describe a car by saying that "its color is red and it has two seats", we can represent that car with a JavaScript object with two properties: 

car = {         // start of object definition
  color: "red", // a comma is needed between properties
  seatCount: 2  // properties can span multiple lines
};              // end of object definition

Note that an object starts with "{" and ends with "}", with properties separated by commas (,).

You can read or write object properties in two ways:

objectName.propertyName

or

objectName["propertyName"]

So for our car object, you can read or change its color using car.color or car["color"].

There are a couple of key facts to remember about objects:

  • An object can be empty, as represented by {}.
  • You can modify an object's properties: 
    car.color = "blue";
  • The value of a property can be another object itself. For example, we can add a property "wheel" to the car object, and the wheel object can be have 2 properties of its own: "diameter" and "color": 
    car.wheel = {
    diameter: 17, color: "silver"
};

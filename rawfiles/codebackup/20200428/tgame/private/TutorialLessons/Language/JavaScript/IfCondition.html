A conditional statement is used to run certain code statement only when a particular condition is satisfied. 

if (condition) {
  // code to run when condition is true
} else {
  // code to run when condition is false
}

Here "condition" can be a variable or an expression, and its value is either "true" or "false". It is usually a comparison statement using one of the following comparison operations: equal to ( == ), not equal to ( != ), greater than ( > ), less than ( < ), greater than or equal to ( >= ), less than or equal to ( <= ).

For example, you can write: 

if (number1 > number2) {
  // some code to run when number1 is greater than number2
} else {
  // some code to run when number1 is not greater than number2
}

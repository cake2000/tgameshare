In JavaScript, variables are containers for storing data values, such as numbers, strings, or objects. All variables are identified with unique names, which contain letters, digits, underscores and dollar signs, and a valid name must begin with a letter. You have to declare a variable before using it, and the variable names are also case-sensitive.

In Scratch, you can define a variable using .

There are two ways to declare a new variable in JavaScript: "const aimPoint" or "let aimPoint". The key difference is that const variables can't be reassigned to some other value, but variables defined with let don't have that restriction. For example: 

let a = 3; // define a new variable using 'let'
a = 4; // updating a's value is fine

const b = 3; // define a new variable using 'const'
b = 4; // error! can't change a constant's value

After the declaration, we use the equal sign ( = ) to assign a value to the variable. There are also other operators that can be used to assign values, as shown in the table below. Note that we'll be using "++" a lot in future tutorials.

Quiz: Out of the four choices below, which one is NOT a correct variable definition?

  • A. 
    int a = 5;
  • B. 
    const a = 5;
  • C. 
    let a = 5;
  • D. 
    let a = {x: 5, y: 9};

Note that you can not reassign a "const" object to some other object, but you can still add or change that object's properties. For example, if we define "const A = {x: 3}", then we can not reassign A like this "A = {y: 4}", but we can change A like this "A.y = 4".

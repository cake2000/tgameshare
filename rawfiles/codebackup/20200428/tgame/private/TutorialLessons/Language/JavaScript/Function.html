A function is a block of code created to perform a particular task or calculate a value. It can take some optional input parameters and (optionally) return some object or value as output.

A function definition contains 4 parts: 1) the function keyword, 2) the function's name, 3) a list of input parameters enclosed by a paif of "()", and 4) some code statements enclosed by a pair of "{}".

Function names can only contain letters, digits, underscores, and dollar signs. The names are case-sensitive (e.g. getTotal is different from gettotal).

Sometimes a function needs to return some information to the user of the function. That's what a return statement does.

For instance, the following function adds 2 to the input number a, and then returns the result. 

function plus2(a) {
  return a + 2;
}

A JavaScript function runs (performs a task) when you call it by appending a parameter list (can be an empty list) to it, like this: 

// define the plus2 function with one input parameter      
function plus2(a) {
  return a + 2;
}
        
// call the "plus2" function with a parameter of 3 
x = plus2(3); // "x" will take the value 5

Note that the number 3 is assigned to the variable a inside the plus2 function, so it returns the value of 3 + 2 as the output. And if you input 7 to the function, then a is set to 7, and the function plus2 will return 9.

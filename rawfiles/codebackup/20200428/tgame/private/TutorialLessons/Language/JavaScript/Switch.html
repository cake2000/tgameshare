The JavaScript switch statement is used to execute different set of statements based on the values of the expression it evaluates. The syntax of switch statement is as below. 

switch(expression) {
  case x:
    code block 1
    break;
  case y:
    code block 2
    break;
  default:
    code block 3
}

The switch statement evaluates the given expression wrapped in "()", matching the value to a case clause, and executes the code block associated with that case. If there is no matched value in any case clause, the default clause will be executed if provided; if not, no code will be executed. The code block inside the case clause is executed until either the end of the switch statement or a break.

The above switch statement can be translated into a nested if-else statement as below. 

if (expression === x) {
  code block 1
} else if (expression === y) {
  code block 2
} else {
  code block 3
}

If there are multiple case clauses matched the value of the given expression, the first case clause will be executed. Use break to stop the execution, otherwise it goes to the end of switch statement. Sometimes you will want multipe case clauses to use the same code block, then you can put these case statements together. Such as the code below, if a is 0 or 1, code block 1 will get executed. 

switch(a) {
  case 0:
  case 1:
    code block 1
    break;
  case 2:
  case 3:
    code block 2
    break;
  defualt:
    code block 3
}

Here is another example of switch statement. The default clause is optional. In the example below, no default price is set. Since "peach" and "pear" clauses have the same price value, we can put them together without a break in between.

So far your robot has been using a default strength of 30, which means 30% of maximum strength. You can change the strength using a third property strength in the object returned by the getBreakShot function, as shown in the Robot Code editor. The maximum of strength values is 100, and minimum is 2. Any values outside this range will be reset to the maximum or minimum value.

Now let me introduce the last 2 properties cueballx and cuebally for break shot, which specifies where you want to place the cue ball before the break shot. For example, this command will place the cueball at lower left corner of the table:

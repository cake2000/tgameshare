When a pool game begins, the game engine will ask your robot "hey, how do you want to take the break shot?", and your robot replies with "let's aim at this point (...) on the table".

To make that happen, your robot code needs to contain a function called "getBreakShot". This is like telling the world "hey guys, I know how to take a break shot, just ask me!". A function is a chunk of code for performing a task, and in this case the getBreakShot function specifies a shot command.

Then when it's time for your robot to take a break shot, the game engine will retrieve your robot's shot command by calling its getBreakShot function like this: "command = robot.getBreakShot()".

Do you want to learn more about functions or continue with the main tutorial for now?

As shown in your robot code editor on right, I have added a getBreakShot function for you that returns an object with 2 properties: aimx and aimy. Currently both aimx and aimy are set to 0, which essentially tells the game engine to aim at the center point of the table. I'll talk about how to set aimx/aimy values very soon.


Now please click the button on top right to see this function in action!

function getBreakShot() { return { aimx: 0, aimy: 0 // table center }; }

If you are new to JavaScript, an object is a collection of "property: value" pairs wrapped by "{}". Do you want to learn more about objects or continue with the main tutorial for now?

So what values to assign to aimx and aimy? They are coordinates used to specify the aiming point. The game engine will shoot the white cue ball towards this aiming point when executing your shot command.

Each coordinate has 2 parts: the x coordinate tells us the horizontal (left to right) position of a point, and the y coordinate tells the vertical (top to bottom) position of a point. Note that the y coordinate increases as you move down the table.

The pink dot shows the aiming point on the table. Try clicking on different points on the table. You can see the coordinates change as you move the aiming point, and also the trajectory of ball movements.

As shown below, the table is 2000 units wide and 1000 units high, so the x coordinate of any point on table is between -1000 and 1000, and y coordinate between -500 and 500. For example, the table center's coordinate is (0, 0), and the initial position of the cue ball is (-500, 0) (center of table's left half).

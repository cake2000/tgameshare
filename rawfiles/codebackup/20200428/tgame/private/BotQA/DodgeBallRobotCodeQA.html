In the Dodge Ball game, your robot calls the sendCommand function to send commands to the game engine to control your car. The following commands are supported:

Cmd Description Equivalent Manual Input
F Speed up car forward Press the up arrow key
NF Stop speeding up forward Release the up arrow key
B Speed up car backward Press the down arrow key
NB Stop speeding up backward Release the down arrow key
L Turn left until max angle Press the left arrow key
NL Stop turning left Release the left arrow key
R Turn right until max angle Press the right arrow key
NR Stop turning right Release the right arrow key
S Shoot ball forward Press the SPACE key

MyInfo is a read-only reference object for your robot to find out information about the car. Below is a full list of attributes attached to MyInfo:

Attribute Description
x Horizontal position in top-down view, with x=0 being the center of the arena.
z Vertical position in top-down view, with x=0 being the center of the arena.
angle Car's forward direction in top-down view in degrees, with 0 degree pointing to the right on the screen, and 90 degrees pointing up.
hasCapturedBall Whether the car has a ball with it
myBallColor Color index of my ball in preview image
myCarColor Color index of my car in preview image
opponentBallColor Color index of opponent's ball in preview image
opponentCarColor Color index of opponent's car in preview image
